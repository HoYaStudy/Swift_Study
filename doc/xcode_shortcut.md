# Xcode Shortcut

## View

* Show Library : `<SHIFT><CMD><L>`

## Editor

* Toggle Canvas : `<OPT><CMD><RETURN>`
* Refresh Preview : `<OPT><CMD><P>`

## Source Editor

* Show Action Menu : `<CMD><LMOUSE>`
* Show SwiftUI Inspector : `<CTRL><OPT><LMOUSE>`

## Playground

* Live View : `<OPT><CMD><RETURN>`
